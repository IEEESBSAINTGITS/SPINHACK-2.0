# 🌀 SpinHack 2.0
### The Intercollegiate Web Development Hackathon
**Organized by IEEE Student Branch | Saintgits College of Engineering**

---

## 📌 About SpinHack 2.0

SpinHack 2.0 is an intercollegiate online web development hackathon that challenges participants to build AI-integrated websites within a 4-hour sprint. Each team receives their problem statement through a **Spin the Wheel** mechanism at the start of the event — making it a true test of adaptability, creativity, and technical skill.

## 🛠️ Technology Stack

- **Core Tool:** Generative AI (ChatGPT, Gemini, GitHub Copilot, etc.)
- **Refinement:** HTML, CSS, JavaScript
- **Submission Platform:** GitHub

---

## 🏗️ Event Structure

The hackathon is divided into three key phases:

1. **Planning** — Understand the problem statement and strategize your approach
2. **Development** — Build your AI-powered website using the permitted tools
3. **Submission** — Push your final project to the provided GitHub repository

---

## 📤 Submission Guidelines

All teams must submit their work through their designated **private GitHub repository**. The repository must include:

- ✅ An **Abstract** (max 1 page) summarizing the project, approach, key findings, and the Generative AI tools used *(minimum 3 tools)*
- ✅ A **PowerPoint Presentation (PPT)** explaining the problem, solution, and implementation
- ✅ All **project source files** with regular commits throughout the event

> ⚠️ Late submissions will **not** be considered for judgment.

---

## ⚖️ Judging Criteria

Projects will be evaluated by a **panel of faculty judges** on the following parameters:

| Criteria                    | Description                                      |
|-----------------------------|--------------------------------------------------|
| Creativity & Innovation     | Originality and uniqueness of the solution       |
| Design & Aesthetics         | Visual appeal and user interface quality         |
| Functionality & Usability   | How well the website works and user experience   |
| Use of Generative AI Tools  | Effective and meaningful integration of AI       |
| Relevance to Theme          | Alignment with the assigned problem statement    |
| Technical Execution         | Code quality, structure, and performance         |
| Presentation & Pitch        | Clarity and effectiveness of the PPT             |
| ⭐ Bonus                    | Exceptional and innovative use of AI             |

> 🔒 **The decisions of the faculty judges and organizers are final and binding.**

## 📋 Rules & Code of Conduct

- All websites must be **original** and developed during the hackathon timeframe.
- **Plagiarism** or copying existing projects will result in immediate disqualification.
- All code must be committed to the provided GitHub repository with **regular commits**.
- Teams may access mentors and technical support but must develop all stages **independently**.
- Open-source libraries and APIs are permitted; **proper credits and citations are mandatory**.
- All participants must adhere to the **IEEE Code of Ethics** and the college's code of conduct.
- Any form of misconduct, dishonesty, or disrespect will result in **instant disqualification**.
- Organizers reserve the right to modify the schedule and rules at any time.
